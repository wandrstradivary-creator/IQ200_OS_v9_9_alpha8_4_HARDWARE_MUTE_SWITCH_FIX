# IQ200 OS v7.5 SINGLE PASS SCAN X10THINK

## Що змінено

Сканування SD перероблено на один прохід по файловій системі.

Було:

```text
1. scan MP3
2. scan FLAC
3. scan WAV
```

Стало:

```text
1. один рекурсивний прохід SD
2. кожен файл класифікується одразу:
   - MP3 -> media DB
   - FLAC -> media DB
   - WAV -> media DB
   - JPG/JPEG/BMP -> art DB
```

## Переваги

- SD читається менше разів.
- Менше FAT/openNextFile операцій.
- Швидше сканування великих карт.
- Менший ризик WDT.
- Лічильники MP3/FLAC/WAV оновлюються одночасно.
- Обкладинки пишуться в `art_###.db` під час того ж проходу.

## Важливо

Для швидкості прибрано пошук `cover.jpg/folder.jpg` через `SD.exists()` для кожного треку. Поки що кожен медіафайл отримує fallback:

```text
CD_DISK_INTERNAL
```

А всі знайдені картинки записуються в `art_###.db` як:

```text
ART|/Music/Album/cover.jpg
```

Наступна версія може додати швидке зіставлення обкладинок з альбомами без повторного повного сканування SD.

RT Audio не змінювався.
