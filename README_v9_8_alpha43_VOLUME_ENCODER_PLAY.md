# IQ200 OS v9.8-alpha43 VOLUME ENCODER PLAY

## Change

The push button on the volume encoder is now a dedicated PLAY key.

- rotate: volume 0..100%;
- press: start the selected/resumed track;
- the Player screen opens automatically;
- pressing while audio is already active does not restart the decoder;
- PLAY is blocked during a database scan;
- previous Mute/Tone action on this button was removed.
