# IQ200 OS v9.9-alpha7.12 — WebRadio Station Browser

- Press NAV to open the station list.
- Rotate NAV to scroll.
- Press NAV again to play the selected station.
- Rotate VOL to change volume; press VOL to stop playback.

The browser uses the station list maintained by the Web editor and queues play
through the existing asynchronous WebRadio service.
