# IQ200 OS v9.9-alpha7.7 — WebRadio Cyrillic

WebRadio now renders UTF-8 Ukrainian and Cyrillic station names and stream titles on the 480x320 TFT.

Supported Ukrainian letters include `І і Ї ї Є є Ґ ґ`, along with the full Cyrillic alphabet. Long station/title strings are clipped to their metadata row. The Web interface already uses UTF-8 and needs no conversion.
