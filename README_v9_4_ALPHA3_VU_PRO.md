# IQ200 OS v9.4-alpha3 VU PRO

Added persistent VU settings in NVS without changing Audio Core.

Commands:
- `vu status`
- `vu style rect|dot|thin`
- `vu fps 10..30`
- `vu peak on|off`
- `vu hold 50..1500`
- `vu decay 1..10`
- `vuseg 4..24`
- `vu save`

Settings are applied live and restored after reboot.
